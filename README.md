### Features:

1. Navigation:
A header with 3 tabs (Home, Tasks, Settings).
Clicking a tab shows its section and hides the others.

2. Home page:
Contains a welcomimg UI and a “get started” button that navigates to tasks page.


3. Tasks page:
A form with an input + add button
Add new tasks dynamically to a list
Each task must have: id + text + done
If a task is done, strikethrough the text.

4. Settings Section:
Input fields that update profile info (name, email).
A user card that containing the lastest updated name and email, and the number of completed tasks.
